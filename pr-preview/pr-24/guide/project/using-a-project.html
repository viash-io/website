Get the template project

To get started, we provide a template project which already contains a few components. First create a new repository by clicking the ‚ÄúUse this template‚Äù button in the viash_project_template repository or clicking the button below.

Use project template

Then clone the repository using the following command.

Run executable

target/docker/demo/remove_comments/remove_comments \
  --input resources_test/file1.tsv \
  --output output/bar.tsv

Run VDSL3 module

nextflow run .
  -main-script target/nextflow/remove_comments/main.nf \
  -profile docker \
  --input resources_test/file*.tsv \
  --publish_dir output1
tree output1

Run pipeline

nextflow run . \
  -main-script workflows/demo_pipeline/main.nf \
  -profile docker \
  --input resources_test/file*.tsv \
  --publishDir output2
tree output2
